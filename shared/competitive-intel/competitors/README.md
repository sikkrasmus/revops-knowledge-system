# Competitor profiles

One file per competitor. Use [`../../../schema/page-templates/shared/competitor.md`](../../../schema/page-templates/shared/competitor.md) as the template.

Phase 0 ships the template and this README. In-production forks fill this directory with actual competitor profiles as closed-deal data accumulates.

## Naming convention

- `<competitor-slug>.md` — lowercase, hyphenated, from the competitor's primary brand name.

## When to create a new competitor profile

- We've been in 3+ head-to-head deals with them in 90 days, or
- They're cited by a target buyer as an active evaluation option, or
- They've raised a material funding round or released a category-level product that warrants awareness.
